## 소개
게임 QA 분야에서 **사원 → 리드 → 파트장**으로 성장하며,   
라이브 서비스 품질 확보를 위한 다양한 테스트 프로세스를 수립하고 자동화 환경을 구축해왔습니다.    
이 저장소에 제가 진행한 QA 관련 자동화 사례와 개인 프로젝트를 정리했습니다.
